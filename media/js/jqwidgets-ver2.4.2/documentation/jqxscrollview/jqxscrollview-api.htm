Properties

Name Type Default
width String 320

Gets or sets the jqxScrollView's width.

Code example

Initialize a jqxScrollView with the width property specified. 

$("#jqxScrollView").jqxScrollView({ width: '250px' });
height String 320

Gets or sets the jqxScrollView's height.

Code example

Initialize a jqxScrollView with the height property specified. 

$("#jqxScrollView").jqxScrollView({ height: '250px' });
disabled Boolean false

Gets or sets the jqxScrollView's disabled property.

Code example

Initialize a jqxScrollView with the disabled property specified. 

$("#jqxScrollView").jqxScrollView({ disabled: true });
buttonsOffset Boolean [0, 0]

Gets or sets the jqxScrollView's buttonsOffset property. This property sets the offset from the default location of the navigation buttons.

Code example

Initialize a jqxScrollView with the buttonsOffset property specified. 

$("#jqxScrollView").jqxScrollView({ buttonsOffset: [30, 30] });
moveThreshold Number 0.5

Gets or sets the jqxScrollView's moveThreshold property. The moveThreshold property specifies how much the user should drag the current element to navigate to next/previous element. Values should be set from 0.1 to 1. 0.5 means 50% of the element's width.

Code example

Initialize a jqxScrollView with the moveThreshold property specified. 

$("#jqxScrollView").jqxScrollView({ moveThreshold: 0.6 });
currentPage Number 0

Gets or sets the jqxScrollView's currentPage property. The currentPage specifies the displayed element.

Code example

Initialize a jqxScrollView with the currentPage property specified. 

$("#jqxScrollView").jqxScrollView({ currentPage: 1 });
animationDuration Number 30

Gets or sets the animationDuration property. Specifies the duration of the animation which starts when the current page is changed.

Code example

Initialize a jqxScrollView with the animationDuration property specified. 

$("#jqxScrollView").jqxScrollView({ animationDuration: 500});
showButtons Boolean true

Gets or sets whether the navigation buttons are visible.

Code example

Initialize a jqxScrollView with the showButtons property specified. 

$("#jqxScrollView").jqxScrollView({ showButtons: false });
bounceEnabled Boolean true

Gets or sets whether the bounce effect is enabled when pages are changed.

Code example

Initialize a jqxScrollView with the bounceEnabled property specified. 

$("#jqxScrollView").jqxScrollView({bounceEnabled: false});
slideShow Boolean true

Indicates whether the slideShow mode is enabled. In this mode, pages are changed automatically in a time interval.

Code example

Initialize a jqxScrollView with the slideShow property specified. 

$("#jqxScrollView").jqxScrollView({slideShow: true});
slideDuration Number 3000

Gets or sets the duration in milliseconds of a time interval. The current page is changed when the time interval is elapsed.

Code example

Initialize a jqxScrollView with the slideDuration property specified. 

$("#jqxScrollView").jqxScrollView({slideDuration: 4000});

Events
pageChanged Event

This event is triggered when the current page is changed.

Code examples

Bind to the pageChanged event by type: jqxScrollView. 


$('#jqxScrollView').bind('pageChanged', function (event) 
{
    var page = event.args.currentPage;
});

Methods

refresh Method

Refreshes the widget.

Code example

Invoke the refresh method. 

$("#jqxScrollView").jqxScrollView('refresh');
changePage Method

Navigates to a page.

Code example

Invoke the changePage method. 

$("#jqxScrollView").jqxScrollView('changePage', 0);
forward Method

Navigates to the next page.

Code example

Invoke the forward method. 

$("#jqxScrollView").jqxScrollView('forward');
back Method

Navigates to the previous page.

Code example

Invoke the back method. 

$("#jqxScrollView").jqxScrollView('back');
