Properties

Name Type Default
content String jqxPopup

Gets or sets the jqxPopup's content.

Code examples

Initialize a jqxPopup with the content property specified. 

$('#jqxPopup').jqxPopup({content: 'Custom String'});
horizontalOffset Number 0

Gets or sets the horizontal offset of the popup.

Code examples

Initialize a jqxPopup with the horizontalOffset property specified. 

$('#jqxPopup').jqxPopup({ horizontalOffset:3 });
verticalOffset Number 0

Gets or sets the vertical offset of the popup.

Code examples

Initialize a jqxPopup with the verticalOffset property specified. 

$('#jqxPopup').jqxPopup({ verticalOffset:3 });
enableAnimation Boolean true

Gets or sets whether the fade animation is enabled.

Code examples

Initialize a jqxPopup with the enableAnimation property specified. 

$('#jqxPopup').jqxPopup({ enableAnimation:true });
showDelay Number 250

Gets or sets the delay of the fade animation when the popup is going to be opened.

Code examples

Initialize a jqxPopup with the showDelay property specified. 

$('#jqxPopup').jqxPopup({ showDelay:250 });
hideDelay Number 300

Gets or sets the delay of the fade animation when the popup is going to be closed.

Code examples

Initialize a jqxPopup with the hideDelay property specified. 

$('#jqxPopup').jqxPopup({ hideDelay:300 });
showHtml Boolean false

Gets or sets whether to treat the 'content' property value as html inside the popup.

Code examples

Initialize a jqxPopup with the showHtml property specified. 

$('#jqxPopup').jqxPopup({ showHtml: false });
disabled Boolean false

Gets whether the popup is disabled.

Code examples

Initialize a jqxPopup with the disabled property specified. 

$('#jqxPopup').jqxPopup({ disabled:false });

Events
shown Event

This event is triggered when the popup is opened.

Code examples

Bind to the shown event by type: jqxPopup. 

$('#jqxPopup').bind('shown', function () { // Some code here. });
closed Event

This event is triggered when the popup is closed.

Code examples

Bind to the shown closed by type: jqxPopup. 

$('#jqxPopup').bind('closed', function () { // Some code here. });

Methods
close Method

Closes the popup.

Code examples

Invoke the close method. 

$('#jqxPopup').jqxPopup('close');
open Method

Opens the popup.

Code examples

Invoke the close method. 

@param String - popup's content.
@param(optional) Number - popup's vertical offset.
@param(optional) Number - popup's horizontal offset.
$('#jqxPopup').jqxPopup('open', 'content');
