Properties

Name Type Default
disabled Boolean false

Enables or disables the jqxButtonGroup.

Code examples

Initialize a jqxButtonGroup with the disabled property specified. 

$('#jqxButtonGroup').jqxButtonGroup({disabled: false });
mode String 'default'

Sets the jqxButtonGroup's mode. Possible values:'checkbox', 'radio', 'default'.

Code examples

Initialize a jqxButtonGroup with the mode property specified. 

$('#jqxButtonGroup').jqxButtonGroup({mode: 'radio' });
enableHover Boolean false

Enables or disabled the highlight state.

Code examples

Initialize a jqxButtonGroup with the enableHover property specified. 

$('#jqxButtonGroup').jqxButtonGroup({enableHover: true });

Events
selected Event

This event is triggered when a button is selected - in checkboxes or radio buttons mode.

Code examples

Bind to the selected event by type: jqxButtonGroup. 

$('#jqxButtonGroup').bind('selected', function () { // Some code here. });
unselected Event

This event is triggered when a button is unselected - in checkboxes or radio buttons mode.

Code examples

Bind to the unselected event by type: jqxButtonGroup. 

$('#jqxButtonGroup').bind('unselected', function () { // Some code here. });
buttonclick Event

This event is triggered when a button is clicked.

Code examples

Bind to the buttonclick event by type: jqxButtonGroup. 

$('#jqxButtonGroup').bind('buttonclick', function () { // Some code here. });

Methods
setSelection Method

Selects a button.

Code examples

Invoke the setSelection method. 

$('#jqxButton').jqxButtonGroup('setSelection', 0);
getSelection Method

Gets the selected button.

Code examples

Invoke the getSelection method. 

var btn = $('#jqxButton').jqxButtonGroup('getSelection');
enableAt Method

Enables a button at specific index.

Code examples

Invoke the enableAt method. 

$('#jqxButton').jqxButtonGroup('enableAt', 0);
disableAt Method

Disables a button at specific index.

Code examples

Invoke the disableAt method. 

$('#jqxButton').jqxButtonGroup('disableAt', 0);
enable Method

Enables the jqxButtonGroup.

Code examples

Invoke the enable method. 

$('#jqxButton').jqxButtonGroup('enable');
disable Method

Disables jqxButtonGroup.

Code examples

Invoke the disable method. 

$('#jqxButton').jqxButtonGroup('disable');
