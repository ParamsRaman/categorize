Properties

Name Type Default
disabled Boolean false

Enables or disables the dropDownButton.

Code examples

Initialize a dropDownButton with the disabled property specified. 

$('#jqxDropDownButton').jqxDropDownButton({disabled: false });
height Number/String null

Sets the button height.

Code examples

Initialize a button with the height property specified. 

$('#jqxDropDownButton').jqxDropDownButton({ height: '25px' });
width Number/String null

Sets the button width.

Code examples

Initialize a button with the width property specified. 

$('#jqxDropDownButton').jqxDropDownButton({ width: '250px'});
enableBrowserBoundsDetection Boolean false

Code example

Initialize a jqxDropDownButton with the enableBrowserBoundsDetection property specified. When this property is set to true, the popup may open above the button, if there's not enough available space below the button. 

$('#jqxDropDownButton').jqxDropDownButton({ enableBrowserBoundsDetection: false});
autoOpen Boolean false

Gets or sets whether the DropDown is automatically opened when the mouse cursor is moved over the widget.

Code example

Initialize a jqxDropDownButton with the autoOpen property specified. 

$("#dropdownbutton").jqxDropDownButton({ autoOpen: true });
dropDownHorizontalAlignment Boolean 'left'

Sets the DropDown's alignment. Possible values: 'left' and 'right'.

Code example

$('#dropdownbutton').jqxDropDownButton({ dropDownHorizontalAlignment: 'right'});
openDelay Number 350

Sets the delay of the 'open' animation.

Code example

Set the openDelay property 


$('#button').jqxDropDownButton({openDelay: 200});
closeDelay Number 400

Sets the delay of the 'close' animation.

Code example

Set the closeDelay property 


$('#button').jqxDropDownButton({closeDelay: 200});
animationType String 'slide'

Sets the type of the animation. Possible values: 'fade', 'slide' and 'none'.

Code example

Set the animationType property 


$('#button').jqxDropDownButton({animationType: 'none'});

Events
open Event

This event is triggered when the button's popup is opened.

Code examples

Bind to the open event by type: jqxDropDownButton. 

$('#jqxDropDownButton').bind('open', function () { // Some code here. });
close Event

This event is triggered when the button's popup is closed.

Code examples

Bind to the close event by type: jqxDropDownButton. 

$('#jqxDropDownButton').bind('close', function () { // Some code here. });

Methods
setContent Method

Sets the button's content.

Code examples

Invoke the setContent method. 

$('#jqxDropDownButton').jqxDropDownButton('setContent', 'my content');
getContent Method

Gets the button's content.

Code examples

Invoke the getContent method. 

var content = $('#jqxDropDownButton').jqxDropDownButton('getContent');
open Method

Shows the button's content.

Code examples

Invoke the showDropDown method. 

$('#jqxDropDownButton').jqxDropDownButton('open');
close Method

Hides the button's content.

Code examples

Invoke the hideDropDown method. 

$('#jqxDropDownButton').jqxDropDownButton('close');
