Properties

Name Type Default
value Number 0

Gets or sets the progress bar's value The value should be set between min(default value: 0) and max(default value: 100).

Code examples

Initialize a jqxProgressBar with the value property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, value: 100});
max Number 100

Sets the progress max value.

Code examples

Initialize a jqxProgressBar with the max property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, max: 150 });
min Number 0

Sets the progress min value.

Code examples

Initialize a jqxProgressBar with the min property specified. 

$('#jqxProgressBar').jqxProgressBar({ width: 200, height: 25, min: 10 });
orientation String 'horizontal'

Gets or sets the orientation.

Code examples

Initialize a jqxProgressBar with the orientation property specified. 

$('#jqxProgressBar').jqxProgressBar({orientation: "vertical"  });
width Number/String null

Gets or sets the progress bar width.

Code examples

Initialize a jqxProgressBar with the width property specified. 

$('#jqxProgressBar').jqxProgressBar({width:"250px"  });
height Number/String null

Gets or sets the progress bar height.

Code examples

Initialize a jqxProgressBar with the height property specified. 

$('#jqxProgressBar').jqxProgressBar({height:"100px"  });
showText Boolean false

Gets or sets the visibility of the progress bar's percentage's text.

Code examples

Initialize a jqxProgressBar with the showText property specified. 

$('#jqxProgressBar').jqxProgressBar({showText:true});

Events
valuechanged Event

This event is triggered when the value is changed.

Code examples

Bind to the valuechanged event by type: jqxProgressBar. 

$('#jqxProgressBar').bind('valuechanged', function () { // Some code here. });
invalidvalue Event

This event is triggered when the user enters an invalid value( value which is not Number or is out of the min - max range. )

Code examples

Bind to the invalidvalue event by type: jqxProgressBar. 

$('#jqxProgressBar').bind('invalidvalue', function (event) { // Some code here. });
complete Event

This event is triggered when the value is equal to the max. value.

Code examples

Bind to the complete event by type: jqxProgressBar. 

$('#jqxProgressBar').bind('complete', function (event) { // Some code here. });

Methods
actualValue Method

Sets the progress bar's value

Code examples

Invoke the actualValue method. 

//@param Number
$('#jqxProgressBar').jqxProgressBar('actualValue', 50);
