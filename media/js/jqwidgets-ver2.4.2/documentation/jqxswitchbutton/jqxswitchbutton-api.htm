Properties

Name Type Default
width Number/String null

Sets the width.

Code examples

Initialize a jqxSwitchButton with the width property specified. 

$('#jqxSwitchButton').jqxSwitchButton({width:"200px"});
height Number/String null

Sets the height.

Code examples

Initialize a jqxSwitchButton with the height property specified. 

$('#jqxSwitchButton').jqxSwitchButton({height:"400px"});
disabled Boolean false

Gets whether the switch button is disabled.

Code examples

Initialize a jqxSwitchButton with the disabled property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ disabled:true });
orientation String 'horizontal'

Sets the orientation.

Code examples

Initialize a jqxSwitchButton with the orientation property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ orientation:vertical });
onLabel String 'On'

Sets the string displayed when the button is checked.

Code examples

Initialize a jqxSwitchButton with the onLabel property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ onLabel:'ON' });
offLabel String 'Off'

Sets the string displayed when the button is unchecked.

Code examples

Initialize a jqxSwitchButton with the offLabel property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ offLabel:'OFF' });
thumbSize String '40%'

Sets the size of the thumb in percentages.

Code examples

Initialize a jqxSwitchButton with the thumbSize property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ thumbSize:'50%' });
checked Boolean false

Gets or sets the ckeck state. Possible Values: true, false.

Code examples

Initialize a jqxSwitchButton with the checked property specified. 

$('#jqxSwitchButton').jqxSwitchButton({ checked:true });

Events
checked Event

'Checked' event is triggered when the switch button is checked.

Code examples

Bind to the checked event by type: jqxSwitchButton. 

$('#jqxSwitchButton').bind('checked', function (event) { // Some code here. });
unchecked Event

'Unchecked' event is triggered when the switch button is unchecked.

Code examples

Bind to the unchecked event by type: jqxSwitchButton. 

$('#jqxSwitchButton').bind('unchecked', function (event) { // Some code here. });
change Event

'Change' is triggered when the switch button's state changes from one state to another.

Code examples

Bind to the change event by type: jqxSwitchButton. 

$('#jqxSwitchButton').bind('change', function (event) { // Some code here. });

Methods
check Method

Checks the ckeckbox.

Code example

Invoke the check method. 

$('#jqxSwitchButton').jqxSwitchButton('check');
uncheck Method

Unchecks the button.

Code example

Invoke the uncheck method. 

$('#jqxSwitchButton').jqxSwitchButton('uncheck');
toggle Method

Toggles the check state. This method is automatically called when the user clicks the switch button.

Code example

Invoke the toggle method. 

$('#jqxSwitchButton').jqxSwitchButton('toggle');
disable Method

Disables the switch button.

Code example

Invoke the disable method. 

$('#jqxSwitchButton').jqxSwitchButton('disable');
enable Method

Enables the switch button.

Code example

Invoke the enable method. 

$('#jqxSwitchButton').jqxSwitchButton('enable');
