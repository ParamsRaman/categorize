Properties

Name Type Default
filterCallback Function 
                    
function (text, searchValue) {
    return text.toString().toLowerCase().indexOf(searchValue.toLowerCase()) >= 0;
};

Used for filtering the jqxListMenu using the filter input.

Code example

Initialize a jqxListMenu with the filterCallback property specified. 

$('#jqxListMenu').jqxListMenu({ filterCallback: function (text, searchValue) {
    return text.toString().toLowerCase().indexOf(searchValue.toLowerCase()) >= 0;
}});
width String 100%

Gets or sets the jqxListMenu's width.

Code example

Initialize a jqxListMenu with the width property specified. 

$("#jqxListMenu").jqxListMenu({ width: '250px' });
height String auto

Gets or sets the jqxListMenu's height.

Code example

Initialize a jqxListMenu with the height property specified. 

$("#jqxListMenu").jqxListMenu({ height: '250px' });
disabled Boolean false

Gets or sets the jqxListMenu's disabled property.

Code example

Initialize a jqxListMenu with the disabled property specified. 

$("#jqxListMenu").jqxListMenu({ disabled: true });
roundedCorners Boolean true

Gets or sets the jqxListMenu's roundedCorners property.

Code example

Initialize a jqxListMenu with the roundedCorners property specified. 

$("#jqxListMenu").jqxListMenu({ roundedCorners: false });
showNavigationArrows Boolean true

Gets or sets the jqxListMenu's showNavigationArrows property. The showNavigationArrows specifies whether navigation arrows are displayed only for list items with nested Lists

Code example

Initialize a jqxListMenu with the showNavigationArrows property specified. 

$("#jqxListMenu").jqxListMenu({ showNavigationArrows: false });
alwaysShowNavigationArrows Boolean false

Gets or sets the jqxListMenu's alwaysShowNavigationArrows property. The alwaysShowNavigationArrows specifies whether navigation arrows are displayed for all items.

Code example

Initialize a jqxListMenu with the alwaysShowNavigationArrows property specified. 

$("#jqxListMenu").jqxListMenu({ alwaysShowNavigationArrows: true });
placeHolder String 'Filter list items...'

Gets or sets the filter input field's place holder.

Code example

Initialize a jqxListMenu with the placeHolder property specified. 

$("#jqxListMenu").jqxListMenu({ placeHolder: 'Search items...' });
showFilter Boolean false

Gets or sets whether the filter input field is visible.

Code example

Initialize a jqxListMenu with the showFilter property specified. 

$("#jqxListMenu").jqxListMenu({ showFilter: true });
showHeader Boolean true

Gets or sets whether the header will be visible.

Code example

Initialize a jqxListMenu with the showHeader property specified. 

$("#jqxListMenu").jqxListMenu({showHeader: false});
showBackButton Boolean true

Indicates whether the back button will be visible.

Code example

Initialize a jqxListMenu with the showBackButton property specified. 

$("#jqxListMenu").jqxListMenu({showBackButton: true});
backLabel String "Back"

Gets or sets the BackButton's label.

Code example

Initialize a jqxListMenu with the backLabel property specified. 

$("#jqxListMenu").jqxListMenu({backLabel: 'Back'});
animationType String "slide"

Sets or gets the animation's type. There are two possible values for this property: 'slide', 'fade'.

Code example

Initialize a jqxListMenu with the animationType property specified. 

$("#jqxListMenu").jqxListMenu({animationType: 'fade'});
animationDuration Number 250

Sets or gets the animation duration in milliseconds.

Code example

Initialize a jqxListMenu with the animationDuration property specified. 

$("#jqxListMenu").jqxListMenu({animationDuration: 130});
headerAnimationDuration Number 0

Sets or gets the animation duration of the header. Header of a listmenu could be set whether the data-role attribute of a list item is set to 'header'. Example: 

                                    

    <ul data-role="listmenu">
        <li data-role="header">Header</li>
        <li>Item</li>
    </ul>
This property is setting the duration of the animation used for changing the header when different nested listviews are switched. Fade animation is used for switching through different headers.

Code example

Initialize a jqxListMenu with the headerAnimationDuration property specified. 

$("#jqxListMenu").jqxListMenu({ headerAnimationDuration: 250 });
autoSeparators Boolean false

Gets or sets whether auto separators will be generated. Separator is generated for a sequence of list items with equal start character. The property is possible to be set through the attribute data-auto-separators="true". Code Example: 

                                
        
<ul data-role="listmenu" data-auto-separators="true">
            <li>Item 1</li>
            <li>Item 2</li>
</ul>

Code example

Initialize a jqxListMenu with the autoSeparators property specified. 

$("#jqxListMenu").jqxListMenu({ autoSeparators: true });
readOnly Boolean false

Gets or sets whether the list will be read-only. In readOnly mode, items are not clickable.

Code example

Initialize a jqxListMenu with the readOnly property specified. 

$("#jqxListMenu").jqxListMenu({ readOnly: false });
enableScrolling Boolean true

When the jqxListMenu is created with the enableScrolling property set to true, vertical scrollbar automatically appears, if the items overflow the visible area. In order to use this feature, the following files should be included: jqxpanel.js and jqxscrollbar.js.

Methods

changePage Method

Sets the displayed page. The page could be set using a selector or object. The page must be a child of the current list and it's data-role must be set to "listmenu".

Code example

Invoke the changePage method. 

// @param String
$("#jqxListMenu").jqxListMenu('changePage', '#newPage' );
