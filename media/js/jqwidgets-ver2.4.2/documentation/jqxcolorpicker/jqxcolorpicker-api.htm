Properties

Name Type Default
disabled Boolean false

Enables or disables the color picker.

Code examples

Initialize a color picker with the disabled property specified. 

$('#colorPicker').jqxColorPicker({disabled: true });
height Number/String null

Sets the height of the color picker.

Code examples

Initialize a color picker with the height property specified. 

$('#colorPicker').jqxColorPicker({height: 300 });
width Number/String null

Sets the width of the color picker.

Code examples

Initialize a color picker with the width property specified. 

$('#colorPicker').jqxColorPicker({width: 300 });
colorMode String 'saturation'. Possible Values: 'hue', 'saturation'

Sets the color mode.

Code examples

Initialize a color picker with the colorMode property specified. 

$('#colorPicker').jqxColorPicker({colorMode: 'hue' });
showTransparent String false

Sets the showTransparent property.

Code examples

Initialize a color picker with the showTransparent property specified. 

$('#colorPicker').jqxColorPicker({showTransparent: true });

Events
colorchange Event

This event is triggered when a new color is picked.

Code examples

Bind to the colorchange event by type: jqxColorPicker. 


$('#jqxColorPicker').bind('colorchange', function (event) 
{
    var color = event.args;
});

Methods
setColor Method

Sets a color.

Code examples

Invoke the setColor method. 

 $("#jqxColorPicker").jqxColorPicker('setColor', '#bbaaff');
                        or
                         $("#jqxColorPicker").jqxColorPicker('setColor', {r: 150, g: 250, b: 255});
getColor Method

Gets the color.

Code examples

Invoke the getColor method. 

var color = $("#jqxColorPicker").jqxColorPicker('getColor');
