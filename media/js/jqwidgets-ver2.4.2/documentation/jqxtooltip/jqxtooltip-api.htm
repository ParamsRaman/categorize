Properties

Name Type Default
content String jqxTooltip

Gets or sets the jqxTooltip's content.

Code examples

Initialize a jqxTooltip with the content property specified. 

$('#jqxTooltip').jqxTooltip({content: 'Custom String'});
location String 'bottom-right'

Gets or sets the jqxTooltip's location. The location property defines the tooltip's display position. The default value is set to 'bottom-right' which means that when the mouse cursor is over a html element that has an associated tooltip, the tooltip will be displayed at the bottom-right edge of the element.


Possible values: "top", "left", "bottom", "right", "bottom-right", "top-right", "top-left", "absolute", "relative"

Code examples

Initialize a jqxTooltip with the location property specified. 

$('#jqxTooltip').jqxTooltip({content: 'Custom String'});

Events
shown Event

This event is triggered when the tooltip is opened.

Code examples

Bind to the shown event by type: jqxTooltip. 

$('#jqxTooltip').bind('shown', function () { // Some code here. });
closed Event

This event is triggered when the tooltip is closed.

Code examples

Bind to the shown closed by type: jqxTooltip. 

$('#jqxTooltip').bind('closed', function () { // Some code here. });

Methods
close Method

Closes the tooltip.

Code examples

Invoke the close method. 

$('#jqxTooltip').jqxTooltip('close');
open Method

Opens the tooltip.

Code examples

Invoke the open method. 

//@param String - tooltip's content.
//@param(optional) Number - tooltip's vertical offset.
//@param(optional) Number - tooltip's horizontal offset.
$('#jqxTooltip').jqxTooltip('open', 'content');
add Method

Adds a tooltip to html element.

Code examples

Invoke the add method. 

//@param element - html element.
//@param String - tooltip's text.
var element = $('#bottomRight');
$("#jqxTooltip").jqxTooltip('add', element, 'jqxTooltip');
remove Method

Removes a tooltip associated to a html element.

Code examples

Invoke the remove method. 

// @param element - html element.
var element = $('#bottomRight');
$("#jqxTooltip").jqxTooltip('remove', element);
