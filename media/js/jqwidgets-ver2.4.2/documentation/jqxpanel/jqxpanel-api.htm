Properties

Name Type Default
width Number/String null

Sets the panel width.

Code examples

Initialize a jqxPanel with the width property specified. 

$('#jqxPanel').jqxPanel({width:"200px"});
height Number/String null

Sets the panel height.

Code examples

Initialize a jqxPanel with the height property specified. 

$('#jqxPanel').jqxPanel({height:"400px"});
sizeMode String 'fixed'

Sets the sizing mode. In the 'fixed' mode, the panel displays scrollbars, if its content requires it. In the wrap mode, the scrollbars are not displayed and the panel automatically changes its size. Possible Values: 'fixed', 'wrap'.

Code examples

Initialize a jqxPanel with the sizeMode property specified. 

$('#jqxPanel').jqxPanel({sizeMode: "wrap"});
autoUpdate Boolean false

Automatically updates the panel, if its children size is changed.

Code examples

Initialize a jqxPanel with the autoUpdate property specified. 

$('#jqxPanel').jqxPanel({autoUpdate: true});
disabled Boolean false

Gets or sets whether the panel is disabled.

Code examples

Initialize a jqxPanel with the disabled property specified. 

$('#jqxPanel').jqxPanel({ disabled:true });
scrollBarSize Number 17

Gets or sets the scrollbars size.

Code examples

Initialize a jqxPanel with the scrollBarSize property specified. 

$('#jqxPanel').jqxPanel({ scrollBarSize: 20 });

Events
layout Event

Occurs when the layout is performed.

Code examples

Bind to the shown event by type: jqxPanel. 

$('#jqxPanel').bind('layout', function () { // Some code here. });
