Properties

Name Type Default
height Number/String null

Gets or sets width of the scroll bar.

Code examples

Initialize a jqxScrollBar with the height property specified. 

$('#jqxScrollBar').jqxScrollBar({ height: '25px', });
width Number/String null

Gets or sets width of the scroll bar.

Code examples

Initialize a jqxScrollBar with the width property specified. 

$('#jqxScrollBar').jqxScrollBar({ width: '250px' });
vertical Boolean false

Gets or sets the scrollbar's orientation.

Code examples

Initialize a jqxScrollBar with the vertical property specified. 

$('#jqxScrollBar').jqxScrollBar({ vertical: true });
min Number 0

Gets or sets the scrollbar's minimum value.

Code examples

Initialize a jqxScrollBar with the min property specified. 

$('#jqxScrollBar').jqxScrollBar({ min:7 });
max Number 1000

Gets or sets the scrollbar's maximum value.

Code examples

Initialize a jqxScrollBar with the max property specified. 

$('#jqxScrollBar').jqxScrollBar({ max:700 });
value Number

Gets or sets the scrollbar's value.

Code examples

Initialize a jqxScrollBar with the value property specified. 

$('#jqxScrollBar').jqxScrollBar({ value:700 });
step Number 10

Gets or sets the scrollbar's step. The value is increased/decreased with this step when the user presses a scrollbar button.

Code examples

Initialize a jqxScrollBar with the step property specified. 

$('#jqxScrollBar').jqxScrollBar({ step:7 });
largestep Number 50

Gets or sets the scrollbar's largestep. The value is increased/decreased with this largestep when the user presses the left mouse button in the area between a scrollbar button and thumb.

Code examples

Initialize a jqxScrollBar with the largestep property specified. 

$('#jqxScrollBar').jqxScrollBar({ largestep:35 });
thumbMinSize Number 10

Specifies the scrollbar thumb's minimum size.

Code examples

Initialize a jqxScrollBar with the thumbMinSize property specified. 

$('#jqxScrollBar').jqxScrollBar({ thumbMinSize:5 });
showButtons Boolean false

Gets or set whether the scrollbar displays the increase and decrease arrow buttons..

Code examples

Initialize a jqxScrollBar with the showButtons property specified. 

$('#jqxScrollBar').jqxScrollBar({ showButtons: true });
disabled Boolean false

Gets or set whether the scrollbar is disabled.

Code examples

Initialize a jqxScrollBar with the disabled property specified. 

$('#jqxScrollBar').jqxScrollBar({ disabled:true });

Events
valuechanged Event

This event is triggered when the value is changed.

Code examples

Bind to the valuechanged event by type: jqxScrollBar. 

$('#jqxScrollBar').bind('valuechanged', function () { // Some code here.});

Methods
setPosition Method

Sets the thumb's position

Code examples

Invoke the setPosition method. 

//@param Number
$('#jqxScrollBar').jqxScrollBar('setPosition', 50);
